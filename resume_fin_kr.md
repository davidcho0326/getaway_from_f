## 주요 프로젝트

**Context Architect Training -- 교육 기반 세일즈의 실전** (2025--현재)
스타트업 C-Level 대상 Claude 중심 AI 내재화 커리큘럼 설계 및 운영: 더플레이컴퍼니 CEO·팀장급 16명 대상 5회 교육, 유니브 CEO 레벨 전용 8강 프로그램, 이외 6개 이상 산업(패션·미디어·통신·광고·K-pop 엔터·자동차R&D)에 걸쳐 컨설팅/교육 제공. 시간이 가장 비싼 경영진의 과제 제출률 90%+ 달성. 별도 패션 회사 대상 컨설팅/교육이 실제 계약으로 전환 -- MCP 기반 이미지 생성 플랫폼(FNF Studio) 도입 계약 체결. 핵심 인사이트: 창업자에게 '벤더'가 아닌 '교육자'로 접근하면 심리적 저항이 사라지고, 3회차부터 회의적 태도가 기술 통합 기획으로 전환 — 콜드콜로는 수개월이 걸릴 신뢰 구축 과정.

**FNF Studio -- 외부 벤더 의존에서 자체 플랫폼 내재화로** (2024--현재)
이미지 캡셔닝 문제를 4단계에 걸쳐 진화: 100% 수동 라벨링(15만 개 규모 불가) → ChatGPT 웹에 이미지 하나씩 입력(일관성 없음) → Gemini 구조화 I/O(정확도 미달) → Claude VLM API + 시스템 프롬프트(90% 정확도) — 매번 실패를 인정하고 밑바닥부터 재설계한 제1원칙 사고의 실전 기록. 이후 외부 AI 이미지 생성 벤더의 한계(높은 이미지당 비용, 브랜드 DNA 인코딩 불가, 느린 이터레이션)를 경험한 뒤 제1원칙에서 시스템을 재설계. 17개 도구, 7,350개 프리셋 조합, 12기준 품질검증 시스템을 갖춘 자체 MCP 기반 플랫폼 구축. 결과: 비용 68-80% 절감, 생성 속도 50% 단축, 1차 시도 품질 통과율 78% -- 10억원 규모 전사 확장의 기술적 기반.

**전사 Knowledge Graph BI** (2025--현재)
5개 브랜드의 매출·SCM·조직 데이터를 37개 의도 노드와 29개 API 도구를 통해 Claude와 연결하는 MCP 서버 설계. 사내 인트라넷 프로덕션 서버로 배포하여 보고 병목 50% 감소, 자연어 기반 비즈니스 인텔리전스 조회를 전사적으로 가능하게 함.

**Content Factory -- 데이터에서 지식으로, Graph RAG 기반 콘텐츠 기획** (2024--현재)
194개 뷰티 영상 인게이지먼트 데이터를 구조화된 지식으로 변환하는 5세대 Knowledge Graph 시스템 구축(프로토타입에서 Neo4j 프로덕션까지 진화). 콘텐츠 기획서를 자동 생성하는 4-Tool 에이전틱 RAG 시스템 설계. 실전에서 입증된 핵심 테제: 자사 데이터를 LLM이 추론할 수 있는 지식 구조로 변환하는 능력이 곧 스타트업의 경쟁 해자.
